던전/레이드 관리 디자인 시안

실제 화면에 반영하지 않고, 브라우저에서 바로 비교해볼 수 있도록 만든 HTML/CSS 전용 미리보기입니다. 각 시안은 레이아웃 흐름과 정보 위계를 다르게 잡았고, 현재 운영 도구에 어떤 방향이 더 잘 맞는지 판단하는 용도로 보시면 됩니다.

시안 1 · 운영툴 정돈형

안정적인 운영 화면

현재 웹 구조를 크게 깨지 않으면서도 헤더, 필터, 목록, 편집 패널의 위계를 더 명확하게 정리하는 방향입니다.

헤더와 필터가 명확하게 분리됨
관리자 테이블과 폼에 익숙한 구조
기존 UI와 연결하기 쉬움
시안 2 · 블리자드 콘솔형

게임 운영툴 분위기 강화

다크 블루 배경과 금색 포인트를 중심으로, 카라잔 세계관에 더 가까운 운영 콘솔 분위기를 만드는 방향입니다.

게임/GM 도구 느낌이 강함
현재 사이트 분위기와 잘 어울림
시각적 개성이 가장 강함
시안 3 · 분할 워크스페이스형

검색-목록-편집 생산성 중심

좌측에서 항목을 빠르게 찾고 우측에서 상세를 바로 편집하는 구조로, 실제 운영 효율을 우선하는 방향입니다.

목록과 상세를 동시에 다루기 쉬움
작업 흐름이 가장 빠름
운영자가 오래 써도 덜 피로한 구조
권장 흐름: 전체 개편은 시안 1, 던전/레이드 같은 운영형 화면은 시안 3, 분위기 강조는 시안 2가 잘 맞습니다.
