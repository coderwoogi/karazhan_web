던전/레이드 관리 디자인 시안

실제 화면에는 반영하지 않고, 브라우저에서 비교만 할 수 있도록 만든 HTML/CSS 전용 미리보기입니다. 이번에는 단순히 배치만 다른 수준이 아니라, 운영툴형, 콘솔형, 문서형, 작전실형처럼 분위기와 정보 위계가 확실히 다른 안들로 준비했습니다.

시안 1 · 운영툴 정돈형

안정적인 운영 화면

현재 웹 구조를 크게 깨지 않으면서도 헤더, 필터, 목록, 편집 패널의 위계를 더 명확하게 정리하는 방향입니다.

헤더와 필터가 명확하게 분리됨
관리자 테이블과 폼에 익숙한 구조
기존 UI와 연결하기 쉬움
시안 2 · 블리자드 콘솔형

게임 운영툴 분위기 강화

다크 블루 배경과 금색 포인트를 중심으로, 카라잔 세계관에 더 가까운 운영 콘솔 분위기를 만드는 방향입니다.

게임/GM 도구 느낌이 강함
현재 사이트 분위기와 잘 어울림
시각적 개성이 가장 강함
시안 3 · 분할 워크스페이스형

검색-목록-편집 생산성 중심

좌측에서 항목을 빠르게 찾고 우측에서 상세를 바로 편집하는 구조로, 실제 운영 효율을 우선하는 방향입니다.

목록과 상세를 동시에 다루기 쉬움
작업 흐름이 가장 빠름
운영자가 오래 써도 덜 피로한 구조
시안 4 · 전술 지휘실형

작전실 같은 몰입감

어두운 배경과 발광 포인트, 지휘실 같은 대시보드 흐름으로 즉시 상황 파악에 강한 방향입니다.

실시간 운영 느낌이 강함
상단 대시보드와 상세 편집의 대비가 큼
던전/레이드 화면에 개성이 잘 살아남
시안 5 · 에디토리얼 보드형

차분하고 읽기 쉬운 화면

설명과 운영 문맥이 중요한 화면에 맞게 문서형 정보 위계와 여백 중심으로 정리한 방향입니다.

가독성과 텍스트 읽기 편의성이 높음
긴 브리핑, 운영 팁, 설명에 유리
카드보다 문서형 구성에 가까움
시안 6 · 아틀라스 지도형

맵 중심 탐색 구조

던전/레이드 자체를 하나의 지역처럼 다루면서, 미션 목록과 편집 화면을 자연스럽게 연결하는 방향입니다.

맵 기반 콘텐츠 관리와 잘 어울림
탐색과 편집의 흐름이 직관적임
기획서형 관리자 도구 느낌이 남
추천 기준: 가장 안전한 방향은 시안 1, 가장 개성이 강한 방향은 시안 2와 4, 운영 효율 중심은 시안 3, 설명과 문맥이 중요한 화면은 시안 5, 맵 중심 흐름은 시안 6이 잘 맞습니다.
