드랍 아이템 추가

입력 항목 안내 아이템 번호는 DB의 Item Entry입니다. 루팅 방식은 보통 1을 사용하며, 참조 번호는 다른 드랍 그룹을 참조할 때만 입력합니다.
아이템 번호는 item_template.entry 값입니다. 직접 입력보다 아이템 찾기 버튼 사용을 권장합니다.
0~100 사이로 입력합니다. 예: 5 입력 시 5% 확률입니다.
아이템이 드랍될 때 최소로 지급되는 개수입니다.
아이템이 드랍될 때 최대로 지급되는 개수입니다.
같은 그룹 안에서는 서버 드랍 규칙에 따라 하나만 선택될 수 있습니다. 일반 드랍은 0을 사용합니다.
일반 드랍은 1입니다. 특별한 루팅 조건이 있을 때만 변경합니다.
다른 드랍 묶음을 참조할 때 사용합니다. 일반 아이템 드랍은 0입니다.
예로 설정하면 퀘스트 조건용 드랍으로 처리됩니다.

참조 묶음 연결

참조 묶음 안내 참조 묶음은 reference_loot_template에 저장된 공통 드랍 목록입니다. 여러 크리처가 같은 묶음을 공유할 수 있습니다.
creature_loot_template.Reference 값이며, reference_loot_template.Entry와 연결됩니다.
