Karazhan Main Status Concepts

메인 서버 ON/OFF 표시 UI 시안

카라잔 메인 화면에서 서버 상태를 즉시 확인할 수 있는 UI 후보입니다. 실제 적용 시 API 상태값에 따라 ON/OFF, 접속자 수, 점검 문구를 자동으로 바꾸는 구조로 연결할 수 있습니다.

1. 히어로 배지형

메인 첫 화면의 핵심 CTA 위에 서버 상태를 강하게 노출하는 방식입니다.

WORLD SERVER ONLINE
카라잔 서버가 열려 있습니다

현재 접속 가능 상태입니다. 접속 방법을 확인하고 바로 모험을 시작하세요.

2. 운영 현황 카드형

메인 정보 카드 영역에 넣기 좋고, 접속자/지연시간/업데이트 시간을 함께 보여줍니다.

서버 운영 현황
ON
월드 서버정상
현재 접속자128명
갱신방금 전

3. 상단 헤더 미니 배지형

항상 보이는 헤더에 작은 상태 배지를 추가하는 방식입니다. 노출은 작지만 접근성이 좋습니다.

THE KARAZHAN
서버 ON
메인 콘텐츠 영역

4. 월드/어스 분리형

월드 서버와 어스 서버를 각각 보여줘야 할 때 적합합니다.

월드 서버게임 접속 및 플레이
ONLINE
어스 서버보조 서비스 / 테스트 영역
OFFLINE

5. 점검 안내 배너형

OFF 상태일 때 메인에서 점검 사유와 예상 종료 시간을 크게 안내하는 방식입니다.

SERVER OFF
현재 서버 점검 중입니다

더 쾌적한 환경을 위해 서버 점검을 진행하고 있습니다. 예상 종료 시간은 23:30이며, 상황에 따라 변동될 수 있습니다.

6. 컴팩트 리스트형

공지사항 또는 미디어 영역 옆에 작게 붙이기 좋은 상태 목록입니다.

월드 서버
접속 가능
ON
인증 서버
점검 중
OFF
웹 서비스
정상 운영
